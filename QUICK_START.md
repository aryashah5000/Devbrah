# ⚡ Quick Start - 2 Minutes

## Backend (Terminal 1)

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

✅ Backend running at http://localhost:8000

## Frontend (Terminal 2)

```bash
cd frontend
npm install
npm run dev
```

✅ Frontend running at http://localhost:5173
